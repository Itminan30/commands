1. Version check:
	php -v

2. Check all available php versions:
	ls -la /etc/php
	ls /usr/bin/ | grep php

3. Switch PHP version:
	sudo update-alternatives --config php
	sudo update-alternatives --set php /usr/bin/php[version]

4. Install PHP:
	Do Once:
	sudo apt update
	sudo apt install software-properties-common
	sudo add-apt-repository ppa:ondrej/php
	sudo apt update
	
	Mandatory:
	sudo apt install php7.3 php7.3-fpm
	sudo apt install php8.3 php8.3-fpm libapache2-mod-php8.3 php8.3-mysql php8.3-mbstring php8.3-xml php8.3-gd php8.3-simplexml php8.3-zip php8.3-curl php8.3-bcmath php8.3-common php8.3-xmlrpc php8.3-cli php8.3-json

5. Uninstall PHP:
	Version:
	sudo apt remove php7.3
	
	modules:
	sudo apt remove php7.3-*

6 Change php in apache server (mod_php):
	Check the php version( can be checked my phpinfo()): a2query -m | grep -E 'php|proxy_fcgi|setenvif' 
	First disable the current php version: sudo a2dismod php8.3
	Enable the required version: sudo a2enmod php8.3
	Restart the apache server: sudo systemctl restart apache2
